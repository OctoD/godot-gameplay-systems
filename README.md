# How to install

- Download the latest version (both from the asset store or from github)
- Unpack  the zip file
- Copy the `addons` folder to your project folder
- Enjoy.

# How to develop using source code

- Follow these guidelines to install the necessary tooling [https://docs.godotengine.org/en/stable/contributing/development/compiling/index.html#toc-devel-compiling](compiling a gdextension module)
- Clone the repository (with submodules) 
- Run `scons target=release_debug` in the root of the repository
- Run the godot editor and open the project in the `demo` folder
